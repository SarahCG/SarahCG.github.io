Arrays and Hashes

6/30/15

In this post I would like to cover Arrays and Hashes in Ruby.

Arrays


An array a list of objects in ruby. An array can hold numbers, words, and even other arrays.
You can find objects in arrays using their index, the index is the place in the array where a particular item is.
The tricky thing about Arrays is that their index starts at 0, not 1. 
Drinks = ['Hawaiian Punch', 'Hi-C Ecto Cooler', 'Tang', 'Kool-Aid']

Even though Hawaiian Punch is the first item in the Array, it's index is 0. Ecto Cooler's index is 1, Tang's index is 2 and Kool-Aid is 3.
The great thing about using arrays in Ruby is that it remembers the index and the object so if you are looking for something specific this Arrays are the best way to go.

Hashes


A Hash is similar to an Array, it is less of an ordered list, and more of a heap. Hashes are a collection of key and value pairs. You can index the values in a hash by assigning them a key. Key and values can be anything you want.
Drinks ={
'Hawaiian Punch' => 'The red drink'
'Hi-C Ecto Cooler' => 'The green drink'
'Tang' => 'The orange drink'
'Kool-Aid' => 'The drink that destroys walls'}

In this example, the key is the brand (Hawaiian Punch, Hi-C Ecto Cooler, Tang and Kool-Aid) and the value is their description (The red drink, The green drink, The orange drink, and The drink that destroys walls)
If I wanted to find a description of Kool-Aid I could access the key Kool-Aid and get the value The drink that destroys walls.

Array vs. Hash


Arrays are useful if you want to store a bunch of items and have them ordered for you. An example of where an array would come in handy is a submission form that has multiple fields. If you want to pass through the fields quickly to get to the email address Arrays can help you do that. 
submission_form = ['first_name', 'last_name', , 'age', 'email_address']

Hashes are good if you want to associate one item with something else. An example of when you would use Hash would be if you wanted to map out fruit in your fruit bowl, you could easily find out how many apples, oranges, or grapes you have by assigning them keys and values. 
Fruit_bowl = {
'Apples' => 3
'Oranges' => 2
'Grapes' => 0
'Bannanas' => 3
}
